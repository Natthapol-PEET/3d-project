# 3d-project

## Require
Python 3.10.9
pip 22.3.1

## Creating Virtual Environments
python3 -m venv venv

## You may activate it
source venv/bin/activate

## Install package from requirements.txt
pip3 install -r requirements.txt

## How to run script
python3 src/main.py

## To deactivate a virtual environment, type:
deactivate
